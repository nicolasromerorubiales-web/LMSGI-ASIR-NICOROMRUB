Introducción

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto. Es uno de los sistemas de bases de datos más populares y se utiliza en una variedad de aplicaciones, desde sitios web hasta aplicaciones empresariales.

show databases

El comando SHOW DATABASES se utiliza para mostrar todas las bases de datos en el servidor MySQL.
